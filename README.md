<h1>My First JavaScript Project: Fotogram! 📸</h1>

A modern and responsive photo gallery built with **HTML**, **CSS**, and **Vanilla JavaScript**.  
This project was created to practice dynamic rendering, DOM manipulation, and interactive frontend development without using frameworks.

---

<h2>Features</h2>

- Dynamic image gallery
- Fullscreen overlay / modal view
- Previous & next image navigation
- Responsive design for desktop and mobile
- Hover effects and smooth UI interactions
- Close overlay by clicking outside or using the close button
- Current image counter
- Automatic image title rendering

---

<h2>🛠️ Technologies Used</h2>

- HTML
- CSS
- JavaScript

---

<h2>What I Learned</h2>

This project helped me improve my skills in:

- DOM manipulation
- Dynamic rendering with template strings
- Arrays & loops
- Event handling
- `addEventListener()`
- `event.stopPropagation()`
- Responsive web design
- Flexbox & Grid layouts
- Modal handling with `<dialog>`
- Clean project structure
- Keyboard navigation

---

<h2>Goal of the Project</h2>

The goal of this project was to better understand how interactive websites work using pure JavaScript and to strengthen my frontend development fundamentals.

Instead of filling the HTML with static content, almost everything is dynamically rendered through JavaScript.
